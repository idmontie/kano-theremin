# Theremin with Raspberry Pi + Kano Motion Sensor

## Getting Started

Prerequisites:

1. **nvm** is installed
2. If you are Debian/Ubuntu, make sure you have ALSA installed. See [node-speaker docs](https://github.com/TooTallNate/node-speaker).

```sh
npm i
npm start
```
